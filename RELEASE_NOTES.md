# Winlinez Revival 1.0.0

Initial public release of Winlinez Revival, a portable WinLinez / Color Lines style puzzle game for Windows.

## Highlights

- Single-file portable Windows build, no installer or Python required.
- Modern dark interface with Chinese and English UI.
- Classic 9x9 line-clearing gameplay with path-based movement.
- Animated ball movement and selected-ball effects.
- King and Challenger characters with podium movement when the challenger beats the best score.
- In-game help via the `i` button.
- Local best-score storage beside the exe.

## Why This Exists

This remake was built because the old Winlinez no longer runs reliably on newer Windows systems, but the game is still loved at home. The goal is to keep that familiar puzzle experience easy to run and easy to share.

## Portable Package

Use `Winlinez-Revival-portable.zip` from the GitHub release. The archive contains:

- `Winlinez-Revival.exe`
- `README.txt`

The best score is stored as `winlinez_high_score.json` next to the exe after playing.
